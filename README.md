# CareerPilot AI

CareerPilot AI is a modern career management platform designed to help students and job seekers organize their job search process efficiently.

## Features

### Dashboard
- Career progress overview
- Productivity tracking
- Application pipeline visualization
- AI-powered insights

### Application Tracking
- Manage job applications
- Track application status
- Monitor interviews and offers
- Organize job search activities

### ATS Resume Analyzer
- Resume scoring dashboard
- ATS compatibility analysis
- Keyword match evaluation
- Skill gap identification
- AI recommendations

### Cover Letter Generator
- Generate tailored cover letters
- Job description analysis
- Resume-based customization

### Recruiter CRM
- Manage recruiter contacts
- Track communication history
- Follow-up management
- Relationship tracking

### Interview Preparation Hub
- Technical interview questions
- Behavioral interview questions
- HR interview preparation
- Readiness tracking

### Analytics Dashboard
- Application statistics
- Interview conversion tracking
- Performance insights
- Progress visualization

### Settings & Personalization
- Light Theme
- Dark Theme
- System Theme
- Accent color customization
- User preferences
